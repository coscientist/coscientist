## HQ 프론트매터 제목 자동 렌더링

**범위**: `apps/hq`

### 변경 사항

| 파일 | 변경 |
| ---- | ---- |
| `app/[lang]/[[...slug]]/page.tsx` | MDX 콘텐츠 앞에 `<h1>{page.data.title}</h1>` 추가하여 프론트매터에서 제목 자동 렌더링 |
| `content/docs/index.mdx` | 제목이 프론트매터에서 렌더링되므로 중복된 `# H1` 제목 제거 |

### 기술적 결정

1. **단일 소스 원칙**: 기존에는 HQ 페이지에 프론트매터 `title`(메타데이터/사이드바용)과 수동 `# H1`(페이지 표시용) 모두 필요했음. 이로 인해 중복과 유지보수 부담 발생.

2. **자동 렌더링 패턴**: `page.tsx`를 수정하여 MDX 콘텐츠 앞에 프론트매터 제목을 `<h1>` 요소로 렌더링. 이제 작성자는 프론트매터에 `title`만 설정하면 됨.

3. **중복 제거**: 기존에 프론트매터 제목과 H1이 모두 있던 `index.mdx`에서 명시적 `# H1` 제거하여 페이지에 제목 중복 방지.

### 변경 전후 비교

| 변경 전 | 변경 후 |
| ------- | ------- |
| 프론트매터에 `title` 설정 | 프론트매터에 `title` 설정 |
| 콘텐츠에 `# H1` 수동 추가 | (자동) |
| 제목 변경 시 둘 다 수정 | 프론트매터만 수정 |
