Prática de recuperação é fortalecer a memória ao recordar ativamente
informações, em vez de apenas revisá-las passivamente. Cada recuperação
bem-sucedida torna a memória mais fácil de acessar depois, e cada recuperação
malsucedida dá a você um alvo preciso para revisar novamente.

Na prática, isso se parece com simulados, flashcards, ou explicar um conceito a
partir de uma página em branco. [Recordação Ativa](./active-recall) é o
comportamento do dia a dia; [Repetição Espaçada](./spaced-repetition) é uma
forma comum de programá-la. [Intercalação](./interleaving) adiciona outra
dimensão: misturar tipos de problemas dentro de uma sessão para melhorar a
discriminação.
