LLM என்பது இயல்பான மொழியை முன்கணித்து உருவாக்க, அளவில்லா அளவு உரைத் தொகுதிகளில்
பயிற்சி செய்யப்பட்ட நியூரல் நெட்வொர்க் மாதிரிகளை குறிக்கிறது.
எடுத்துக்காட்டுகள்: GPT, Claude, Gemini, மற்றும் Llama. பயிற்சி தரவிலுள்ள
புள்ளியியல் வடிவங்களை கற்றுக்கொள்வதன் மூலம், LLM-கள் சுருக்கம், மொழிபெயர்ப்பு,
கேள்வி-பதில், குறியீடு உருவாக்கம் போன்ற பலவகை மொழிப் பணிகளைச் செய்ய முடியும்.

[இணை-விஞ்ஞானி](./coscientist) இற்காக, LLM-கள்
[பரிசீலனை உழைப்பு](./contemplation-labor)-ஐ செய்யும் இயந்திரம்: கருதுகோள்களை
முன்வைத்தல், ஆதாரங்களைச் சேகரித்தல், எதிர்மாதிரிகளை கண்டுபிடித்தல், மற்றும்
வாதங்களை கட்டமைத்தல். LLM-கள் எந்த மொழியையும் வாசிக்க முடிவதால்,
[மொழித்தாண்டிய ஒருங்கிணைப்பு](./cross-linguistic-synthesis) என்பதைக் இயல்பான
திறனாகவே இயலுமைப்படுத்துகின்றன.

எனினும், LLM-களுக்கு அடிப்படை வரம்புகள் உள்ளன. அவை உண்மைக்காக அல்ல; நம்பத்தகுந்த
அடுத்து டோக்கன்களை உகப்பாக்குவதற்காக செயல்படுகின்றன. அவை
[ஹாலூசினேட்](./hallucination) செய்யக்கூடும்: உண்மைக்கு முரணான தகவலாக
இருந்தாலும், நம்பிக்கையுடன் மற்றும் ஒற்றுமையாகத் தோன்றும் உரையை உருவாக்குதல்.
அவை [சீர்மைச் சிக்கல்](./fluency-trap) -க்கு ஆளாகக் கூடியவை: தவறுகளை மறைக்கும்
மெருகான உரை. அவைகள் ஒரே போன்ற பயிற்சி தரவை பகிர்ந்து கொள்வதால்,
மாதிரிகளுக்கிடையிலான ஒப்புதல் என்பது சுயாதீனமான [சரிபார்ப்பு](./verification)
அல்ல; ஒன்றுக்கொன்று தொடர்புடைய பாகுபாடு -யின் பிரதிபலிப்பாக இருக்கலாம் (பார்க்க:
[ஆதார சுயாதீனம்](./evidence-independence)).

இதனால்தான் [இணை-விஞ்ஞானி](./coscientist), LLM-களை முன்னறிவிப்புத்
தெய்வவாக்குகளாக அல்ல; கருவிகளாகவே கருதுகிறது. [இயக்குநர்](./operator)
இறையாண்மையைத் தக்க வைத்துக் கொள்கிறார்;
[அறிவியற் நெறிமுறை அடுக்கு](./epistemic-protocol-layer)
[தடமறியத்தன்மை](./traceability) மற்றும்
[மறுப்புமுன் தேடல்](./rebuttal-first-search)-ஐ அமல்படுத்துகிறது; மேலும்
[Multi-AI Consensus Protocol](./multi-ai-consensus-protocol) மாதிரிகளின் கருத்து
வேறுபாட்டை நெருக்கமான ஆய்வுக்கான சிக்னலாகப் பயன்படுகிறது. LLM-கள் தேடலும்
கட்டமைப்பும் செய்கின்றன; மனிதர்கள் சரிபார்ப்பும் முடிவெடுப்பும் செய்கின்றனர்.
