El conjunto mínimo de contradicción es la colección más pequeña de premisas,
afirmaciones y relaciones que, en conjunto, generan un conflicto. Es el "nudo
central" del desacuerdo: si puedes resolver o redefinir el alcance de este
conjunto, el conflicto más amplio a menudo se disuelve.

Por eso es el objetivo de la atención humana en
[Estrategias de Exploración](./exploration-strategies). Técnicas como el
[corte mínimo](./minimum-cut) pueden ayudar a sacar a la superficie núcleos
candidatos para que no tengas que leerlo todo para encontrar lo que realmente
importa.
