Une réfutation indirecte est une contestation qui ne vise pas
l’[affirmation](./claim) elle-même, mais les bases sur lesquelles elle repose :
la [méthode](./method), l’[hypothèse](./assumption) ou la définition qui rend
l’affirmation pertinente. Si la réfutation indirecte réussit, l’affirmation perd
son appui même si aucun [contre-exemple](./counterexample) direct n’est trouvé.

Cette distinction compte pour la résolution. Une [attaque](./attack) directe
peut recevoir une réponse par davantage de preuves ; une réfutation indirecte
exige de réexaminer les fondations. Dans les débats, les réfutations indirectes
restent souvent sans réponse parce qu’elles donnent l’impression de « changer de
sujet », mais elles peuvent être décisives.

Dans un [Graphe dialectique](./dialectical-graph), les réfutations indirectes
constituent un type d’arête spécifique consigné dans les
[Arêtes du graphe dialectique](./dialectical-graph-edges). Les suivre séparément
permet au système de distinguer « nous ne sommes pas d’accord sur les faits » de
« nous ne sommes pas d’accord sur la manière dont les faits doivent être
recueillis ».
