Механизмы синтеза — это паттерны и протоколы, которые
[Соисследователь](./coscientist) использует, чтобы сделать несопоставимые
источники сопоставимыми и получать структурированный результат из конфликта.

## Базовый процесс

- [Синтез знаний](./knowledge-synthesis) — выравнивание предпосылок, определений
  и границ рассмотрения, чтобы сделать разногласия явными

## Паттерны ввода

- [Столкновение документов](./document-collision) — принуждение источников к
  взаимодействию с утверждениями друг друга
- [Соперничество](./contention) — структурированный конфликт как объект первого
  класса

## Паттерны вывода

- [Карта разветвлённых разрешений](./branched-resolution-map) — показ нескольких
  валидных выводов на основе разных предпосылок
- [Узел вопроса](./issue-node) — объединение сталкивающихся утверждений для
  фокусного разрешения

## Инструменты анализа

- [Таксономия несовместимости](./incompatibility-taxonomy) — классификация
  причин, по которым источники расходятся во мнениях
- [Минимальный набор противоречий](./minimum-contradiction-set) — нахождение
  ядра конфликта
- [Минимальный разрез](./minimum-cut) — изоляция структуры, порождающей
  противоречие

## Многоязычное измерение

- [Межъязыковой синтез](./cross-linguistic-synthesis) — объединение знаний из
  нескольких языков
- [Языконезависимая индексация](./language-agnostic-indexing) — концептуальный
  индекс, выходящий за пределы конкретных языков
- [Потеря нюансов при переводе](./translation-nuance-loss) — отслеживание того,
  что теряется при переходе между языками
