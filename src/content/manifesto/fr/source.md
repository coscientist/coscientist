Une source est le point d’origine d’un [segment de preuve](./evidence-span) ou
d’une [affirmation](./claim) : un document, un jeu de données, une expérience ou
une personne d’où provient l’information. Les sources ancrent la
[provenance](./provenance) et rendent la
[chaîne de responsabilité](./responsibility-line) traçable.

Toutes les sources ne se valent pas. Une source primaire rapporte des
observations originales ; une source secondaire résume ou interprète ; une
source tertiaire compile à partir de sources secondaires. Chaque étape introduit
un risque de dérive, et [Coscientifique](./coscientist) suit ces distinctions au
moyen d’arêtes explicites `cites` dans le
[Graphe dialectique](./dialectical-graph).

Les sources sont des nœuds du graphe (voir
[Nœuds du graphe dialectique](./dialectical-graph-nodes)) avec des métadonnées
telles que l’auteur, la date et le contexte. Lorsqu’une affirmation est
contestée, la source fournit le point de départ pour un réexamen.
