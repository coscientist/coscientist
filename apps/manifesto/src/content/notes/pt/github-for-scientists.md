---
title: GitHub para Cientistas
description: Visão de tratar experimentos como código versionado e auditável
---

"GitHub para Cientistas" é uma visão na qual experimentos científicos são tratados como código: versionados, repetíveis e auditáveis. Assim como a engenharia de software passou de scripts ad-hoc para integração contínua e controle de versão, a ciência poderia passar de cadernos de laboratório pontuais para pipelines reprodutíveis com rastreamento de proveniência.

Essa visão é central para [ScienceOps](./scienceops). Quando experimentos se tornam artefatos versionados, o ciclo entre hipótese e [verificação](./verification) encurta. Falhas se tornam rastreáveis. Sucessos se tornam replicáveis por outras pessoas sem esforço hercúleo.

[Coscientist](./coscientist) é a camada cognitiva para esse tipo de fluxo de trabalho: gerenciando [alegações](./claim), [evidências](./evidence-span) e refutações em muitos projetos, enquanto preserva a [linha de responsabilidade](./responsibility-line) e apoia o papel de [engenheiro de ciências naturais](./natural-science-engineer).
